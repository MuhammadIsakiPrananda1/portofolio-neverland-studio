# Scripts

Utility shell scripts for development, deployment, and maintenance.

| Script | Purpose |
|--------|---------|
| `setup-port8080.sh` | Configures the dev server to bind on port 8080 (useful behind reverse proxies) |

## Usage

```bash
# Make scripts executable (first run only)
chmod +x scripts/*.sh

# Run a script
./scripts/setup-port8080.sh
```
